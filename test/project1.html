프로젝트 1: AI 기반 일정 관리 앱

이곳에 프로젝트에 대한 한두 문장의 핵심 요약을 적습니다. 예를 들어, "자연어 처리 기술을 활용하여 사용자의 대화형 입력을 분석하고, 자동으로 일정을 등록 및 관리해주는 지능형 웹 애플리케이션입니다." 와 같이 작성할 수 있습니다.

프로젝트 개요

프로젝트를 시작하게 된 계기, 목표, 그리고 어떤 문제를 해결하고자 했는지에 대해 상세히 서술하는 공간입니다. 사용자의 어떤 불편함을 해소하고 싶었는지, 기술적으로 어떤 도전을 하고 싶었는지 등을 자유롭게 작성해 보세요.

주요 기능

  • 대화형 일정 추가: "내일 3시에 강남에서 회의"와 같은 문장을 입력하면 자동으로 시간, 장소, 내용을 파싱하여 일정을 생성합니다.
  • 리마인더 기능: 등록된 일정 시작 전에 사용자에게 이메일 또는 브라우저 알림을 전송합니다.
  • 캘린더 연동: 구글 캘린더와 실시간으로 동기화하여 일정을 통합 관리할 수 있습니다.

사용한 기술

React Node.js Express MongoDB Natural Language API JWT

어려웠던 점 및 해결 과정

프로젝트를 진행하며 겪었던 기술적, 논리적 어려움과 그것을 어떻게 해결했는지 구체적으로 작성하면 좋습니다. 예를 들어, "다양한 형태의 비정형 자연어 문장을 정확하게 분석하는 데 어려움이 있었습니다. 이를 해결하기 위해 정규 표현식과 여러 NLP 라이브러리를 테스트하고, 최종적으로는 Google Natural Language API를 도입하여 인식률을 95%까지 끌어올렸습니다." 와 같이 작성할 수 있습니다.
