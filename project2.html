프로젝트 2: 반응형 이커머스 웹사이트

데스크탑, 태블릿, 모바일 등 모든 디바이스에서 최적의 쇼핑 경험을 제공하는 완전 반응형 이커머스 웹사이트를 기획하고 개발했습니다.

프로젝트 개요

기존의 많은 쇼핑몰들이 모바일 환경에서 사용성이 떨어진다는 문제점에서 출발하여, 모바일 우선(Mobile-First) 접근법으로 사용자 친화적인 UI/UX를 설계하는 데 중점을 두었습니다. 순수 JavaScript와 CSS만으로 동적인 기능을 구현하여 웹 프레임워크의 기본 원리를 깊이 이해하는 것을 목표로 했습니다.

주요 기능

  • 동적 상품 카탈로그: JSON 데이터를 비동기적으로 불러와 상품 목록을 동적으로 렌더링합니다.
  • 장바구니 기능: 사용자가 선택한 상품을 Local Storage에 저장하여, 브라우저를 닫았다 열어도 장바구니가 유지됩니다.
  • 100% 반응형 디자인: CSS 미디어 쿼리를 적극적으로 활용하여 화면 너비에 따라 레이아웃과 폰트 크기가 유연하게 변합니다.

사용한 기술

HTML5 CSS3 (Flexbox & Grid) JavaScript (ES6+) Fetch API Local Storage

어려웠던 점 및 해결 과정

가장 큰 도전은 JavaScript만으로 장바구니의 상태(추가, 삭제, 수량 변경)를 관리하는 것이었습니다. 페이지 새로고침 없이 장바구니 총액과 수량이 모든 관련 영역(헤더, 장바구니 페이지 등)에 실시간으로 반영되게 하기 위해, 상태가 변경될 때마다 특정 이벤트를 발생시키고 이를 수신하여 UI를 업데이트하는 '옵저버 패턴'을 간단하게 구현하여 해결했습니다.
